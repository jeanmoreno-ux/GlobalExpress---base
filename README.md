# Global Express System

Sistema de gestión para operaciones de courier y logística.

## Objetivo
Desarrollar una plataforma para controlar:

- Recepción de paquetes
- Clasificación
- Rutas de despacho
- Entregas
- Control de evidencias
- Dashboard operativo

## Tecnologías previstas

- Backend: Laravel
- Frontend: React
- Base de datos: MySQL
- Infraestructura: GitHub + Hostinger

## Estado del proyecto

Proyecto en fase inicial de desarrollo.
